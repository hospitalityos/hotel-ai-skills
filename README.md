## Installation

### Option 1: Install via Claude (Recommended)

Add this marketplace to your Claude installation:

```bash
claude plugin add --marketplace https://github.com/hospitalityos/hotel-ai-skills
```

Then browse and install individual skills from your Claude dashboard.

### Option 2: Browse & Explore

Visit the skills directory at: **[hospitalityos.tech/skills](https://hospitalityos.tech/skills)**

## Data Integration

The Hotel Intelligence Suite connects to common hospitality platforms:

- **Property Management Systems (PMS):** Opera, Marriott, InnSpire, Stayntouch
- **Revenue Management:** RMS, Rainmaker, Hoist
- **Review Platforms:** Google Maps, TripAdvisor, Booking.com
- **Financial Systems:** QuickBooks, NetSuite, property accounting
- **Labor Management:** Deputy, Toast, ADP

See [CONNECTORS.md](./hotel-intelligence-suite/CONNECTORS.md) for detailed integration steps.
