## Setup

### 1. Install the Plugin

Install from the Anthropic plugin directory or download the `.plugin` file from [hospitalityos.tech/skills](https://hospitalityos.tech/skills).

### 2. Complete Your Property Profile

On first use, any skill will prompt you to complete the Property Profile — a one-time setup that captures your property's identity, room inventory, revenue centers, market segments, comp set, systems, and ownership structure. This profile is referenced by all 10 skills.

### 3. Connect Your Data

Skills use a 3-tier data acquisition model:
- **Tier 1**: Direct API/MCP connections (Gmail, Google Drive, Google Sheets, HubSpot)
- **Tier 2**: Browser-assisted extraction from your existing hotel systems
- **Tier 3**: Manual file upload (Excel, CSV, PDF) with guided instructions

No new system integrations required — the suite works within your existing hotel tech stack.
