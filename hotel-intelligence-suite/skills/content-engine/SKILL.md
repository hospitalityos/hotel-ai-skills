# HospitalityOS™ Hotel Intelligence Suite
Sales & Marketing Content Engine

## Overview
Professional content generation for hospitality sales and marketing teams. Accelerates proposal writing, campaign creation, and promotional material development while maintaining consistent brand voice and property differentiation.

## Data Acquisition Strategy

### Required Inputs (Tier 1: Always Needed)
- **Brand Voice Guidelines** — Stored in memory after first setup
- **Property Fact Sheet** — Room inventory, meeting spaces, F&B outlets, key amenities
- **Target Audience/Segment** — Meeting planners, corporate, leisure, wedding couples, etc.
- **Campaign Objective** — Lead generation, conversion, retention, awareness, seasonal push
- **Content Type Specification** — What deliverable format is needed?

### Data Sources (Tier 2: MCP Tools)
- **HubSpot** — Contact context, deal stage, relationship history, previous interactions
- **Gmail** — Incoming RFP questions, client communication threads, previous correspondence
- **Google Drive** — Brand asset library, past proposals for tone/format matching, property photos

### Research Layer (Tier 3: Browser/Web)
- Property brand extranet for current rates and packages
- Competitor OTA listings for market positioning and benchmark language
- Property website for messaging consistency

### User Description (Tier 4: Fallback)
User provides specific context: property details, audience, campaign goals, tone preferences.

## One-Time Property Setup: Content Database

On first use, help the user build a reusable **Property Content Database** stored in memory. Capture:

- **Property Description** — 50-word, 100-word, and 250-word versions for flexible use
- **Room Type Descriptions** — Standard, deluxe, suite (include bed types, amenities, key features)
- **Meeting Space Specifications** — Theater/classroom/banquet capacity chart, ceiling height, tech capabilities
- **F&B Outlet Profiles** — Name, cuisine type, capacity, signature dishes, setup flexibility
- **Core Amenity List** — Fitness, pool, business center, parking, concierge, etc.
- **Unique Selling Points** — Location, history, renovations, awards, distinctive experiences
- **Testimonial Quotes** — 2-3 curated guest/client quotes for authenticity
- **Awards & Recognition** — Certifications, industry recognition, recent accolades

This database becomes the source material for all content generated by this skill.

## Content Types & Frameworks

### Group Proposals
**Tone:** Professional, specific, ROI-focused. Address the planner's stated needs directly.

Structure:
1. Personalized opening acknowledging their specific requirements
2. Property overview (why this property fits their event)
3. Proposed room block with rate and flexibility
4. Meeting space recommendation with room diagram/setup options
5. F&B menu options with package pricing
6. Concessions and added value (welcome gift, late checkout, etc.)
7. Closing: next steps and contact info

**Deliverable:** 1-2 page proposal with alternative rate options.

### RFP Responses
**Tone:** Thorough, differentiated, service-focused. Answer every question.

Guidelines:
- Answer every RFP question directly and completely
- Differentiate on service excellence and experience, not price alone
- Include property photos where relevant
- Structure mirrors the RFP format exactly
- Highlight unique capabilities that competitors lack
- Provide 1-2 alternative approaches

**Deliverable:** Formal RFP response matching original format.

### Email Campaigns
**Tone:** Segment-appropriate. Always mobile-optimized.

By Segment:
- **Past Guests** — Warmth, recognition of their visit, exclusive offers
- **Prospects** — Aspirational imagery, benefit-driven copy, clear value prop
- **Corporate** — Efficiency, flexibility, group value, partnership language
- **Meeting Planners** — Capability, vendor credibility, ROI focus

Always include:
- 3 subject line options (A/B/C)
- Clear call-to-action (book, inquire, view details)
- Mobile-responsive formatting
- Unsubscribe compliance

**Deliverable:** Email body + 3 subject lines + design notes.

### Social Media Content
**Tone:** Platform-native. Visual + conversational.

By Platform:
- **Instagram** — Visual-first, aspirational, guest experience storytelling. 1-2 sentences max.
- **LinkedIn** — Thought leadership, industry trends, event hosting capability, culture posts.
- **Facebook** — Community, events, seasonal offers, user-generated content celebration.
- **Twitter/X** — News, real-time engagement, quick tips, industry commentary.

All Posts Include:
- 3-5 relevant hashtags
- Platform-native CTAs
- Alt text for images
- Post scheduling notes

**Deliverable:** 5-7 posts across platforms with hashtags and scheduling guidance.

### OTA Listing Optimization
**Tone:** Keyword-rich, competitive, benefit-focused.

Structure:
- Title tag optimization
- Meta description (SEO-focused summary)
- Full property description with differentiation
- Amenity highlighting (match listing field character limits)
- Photo caption optimization
- Competitive positioning notes

Covers: Expedia, Booking.com, Hotels.com, Airbnb (differentiated format per platform).

**Deliverable:** Copy for each OTA with character counts and formatting notes.

### Website Copy
**Tone:** Benefit-driven, scannable, SEO-conscious.

Types:
- **Landing Pages** — Clear value prop, scannable headlines, conversion path
- **Property Pages** — Room/amenity showcases with features → benefits translation
- **Package Pages** — Benefit-led, comparison tables where useful, clear booking path

Always:
- Front-load benefits (not just features)
- Use short sentences and white space
- Include internal linking recommendations
- Provide meta title/description options

**Deliverable:** Page copy + metadata + internal link recommendations.

### Press Releases
**Tone:** Newsworthy, credible, AP-style formatted.

Standard Structure:
- Headline (benefit-driven)
- Location line
- Lede paragraph (who, what, when, where, why)
- Property overview paragraph
- Quote from GM/key leader
- Details/context paragraphs
- Property boilerplate
- Contact info

**Deliverable:** Formatted press release ready to distribute.

### Wedding/Event Packages
**Tone:** Aspirational yet specific. Inclusive and customizable.

Structure:
- Package name and tagline
- Included elements (venue, catering, bar setup, etc.) by tier
- Pricing tiers with customization guidance
- Personalization options
- Timeline and booking process
- Clear CTA to schedule site visit

**Deliverable:** Package brochure outline + 2-3 pricing tier options.

## Brand Voice Calibration

Before drafting content, identify property brand positioning:

| Property Type | Voice Profile |
|---|---|
| **Luxury** | Warm, sophisticated, understated elegance. No exclamation points. "Experience" over "Enjoy." Let quality speak. |
| **Full-Service** | Confident, professional, versatile. Warm but capable. Emphasize breadth and reliability. |
| **Select-Service** | Efficient, friendly, value-conscious. Convenience and consistency as core messages. |
| **Independent/Boutique** | Distinctive, personality-driven, story-first. Lean into what makes the property unique. |

**Always provide 2-3 headline/subject line options** for any content piece.
